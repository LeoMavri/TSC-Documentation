## Terminology

> [!Important]  
>
> Even if this isn't your first time using scripts, I highly recommend reading this section carefully.  


#### Browser Extensions & Script Managers  

| Abbreviation | Full Name        | Description | Link |  
|-------------|-----------------|-------------|------|  
| **TM**      | Tampermonkey     | A popular userscript manager that allows you to run custom JavaScript on websites. | [Website](https://www.tampermonkey.net/) |  
| **VM**      | Violentmonkey    | An open-source alternative to Tampermonkey with similar functionality. | [Website](https://violentmonkey.github.io/) |  
| **GM**      | Greasemonkey     | One of the original userscript managers, but now less commonly used due to changes in Firefox's extension system. | [Wikipedia](https://en.wikipedia.org/wiki/Greasemonkey) |  

> [!Note]
>
> If you're unsure which one to use, install Tampermonkey
> * [Chrome Store](https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)
> * [Firefox Addon Store](https://addons.mozilla.org/en-US/firefox/addon/tampermonkey/)

> [!Warning]
>
> If you're using Google Chrome, make sure you turn on "Developer Mode" under Settings -> Extensions

#### Browsers & Their Variants  

| Name | Description | Notes |  
|-------------|-------------|------|  
| **Firefox** | A privacy-focused web browser. | Includes forks like Waterfox, LibreWolf, Floorp, Zen etc.. |  
| **Chrome*** | A widely used browser developed by Google.  | This includes all Chromium-based browsers, such as Microsoft Edge, Brave, Vivaldi, Opera etc.  |  

\* I am aware Chrome and Chromium are different things, but for the sake of simplicity I'll put them all under one blanket term.

#### TSC Specific  

- **API Key / Key** - A unique identifier used to authenticate requests when accessing an API. In this context, it refers to the API key you registered with TSC.
- **Companion** - User script that shows the TSC spies on the user's page.
